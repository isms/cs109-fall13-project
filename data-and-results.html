CS 109 Project

Data

We use data about marine incidents from the United States Coast Guard, which included both records of actual incidents from the Coast Guard's MISLE database as well as calls to the National Response Center (NRC). This data set is available here. This is a rich data set which displays seasonal trends, correlations with weather, and other interesting features. You can see an excerpt of the data in the figure above, which plots incidents from 2009 through 2012.
