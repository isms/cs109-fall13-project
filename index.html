CS 109 Project

Predicting Where Voyages Go Wrong

About

This is the write-up for a Fall 2013 final project in CS 109 (Data Science) at Harvard University.

We examine where and when marine incidents happen in United States waters. We focus on:

  1. Analysis of patterns within the data set and commonalities of these incidents.
  2. Visualization using tools such as Python's basemap package.
  3. Prediction using machine learning algorithms.

Data and Results

We use data about marine incidents from the United States Coast Guard. You can find out more about this data set and our results here.

C. Hornbaker, S. Merity, I. Slavitt, 2013
